# RailmodelAutonomousDriver

---
## 概要

本プロジェクトは、Arduino MEGAを活用し、鉄道模型の自動運転システムを構築することを目的としている。実際の鉄道では線路をいくつかの区間に区切り、1つの区間に複数の列車が入らないように制御することで列車同士の衝突を防いでいる。このシステムでは実際の鉄道の運行方式を参考に、Nゲージにおいて1つの線路をいくつかの区間に区切り、2つの列車運転を可能とした。サンプルでは6区間2列車としている。

## 主な機能

*   **最大8区間までの閉塞運転対応** : 最大8区間の自動閉塞運転が可能。
*   **自動加減速制御** : PWM制御によるスムーズな停車・発車。
*   **センサー連動** : フォトリフレクタを用いた列車位置の検出。
*   **自動閉塞信号機制御** : ダイナミック点灯による最大8個の信号機制御が可能。

## 課題と今後の拡張予定
### 課題
列車同士に速度差があるとどうしても車間を詰めて停止するというあおり運転になってしまう。これは出力調整により速度差を減らす、または追い抜きができるようにするという仕組みで解決する。

### 注意信号での速度調整実装
注意信号では減速しない仕様になっており、実物と乖離している。

### 列車ごとの個別設定
可変抵抗を4個使用し、列車ごとに個別に最低速度・最高速度を設定する。

### ポイントレール対応
マイコンからポイントを切り替えられるようにする。
実験ではモータードライバTA7291Pを使って50msのパルスで100％の切り替え率を実現している。
電流が1A程度必要なので、電源容量を2A以上にするまたは、キャパシタの追加など、回路の変更、切り替え実行条件ロジックの追加実装が必要。
プログラム暴走時にコイルを焼き切る可能性があるので、キャパシタに必要な電荷をチャージしてそれを使う方式としたほうがよさそう。
ただし、充電時間が必要となるので、連続しての切り替えに弱くなるという欠点がある。
