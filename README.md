# RailmodelAutonomousDriver

Arduino MEGAを使用した、鉄道模型(Nゲージ)の自動運転・制御プロジェクトである。
---

## 概要

本プロジェクトは、Arduino MEGAの豊富なI/Oピンを活用し、鉄道模型の自動運転システムを構築することを目的としている。実際の鉄道では線路をいくつかの区間に区切り、1つの区間に複数の列車が入らないように制御することで列車同士の衝突を防いでいる。このシステムでは実際の鉄道の運行方式を参考に、Nゲージにおいて1つの線路を4つの区間に区切り、2つの列車を運転している。

## 主な機能

*   **閉塞信号機の自動制御**：列車の位置を検知し、進行（緑）・停止（赤）の信号を自動で切り替える。
*   **自動加減速制御**：PWM制御によるスムーズな停車・発車。
*   **センサー連動**：フォトリフレクタを用いた列車位置の検出。

## システム構成 / ハードウェア仕様

### 1. 使用部品・開発環境
*   **マイコンボード**: Arduino MEGA 2560 R3 (ATmega2560)
*   **開発環境**: Arduino IDE 1.8+ / CLI
*   **主要部品**: フォトリフレクタ（位置検知）、モータードライバIC、各種LED、抵抗器

### 2. ピンアサイン（割当てイメージ）
| ピン番号 | I/O | 接続先・用途 | 備考 |
| :--- | :--- | :--- | :--- |
| **D2** | Input | 列車検知センサー 1 | 外部割り込み（INT0）利用 |
| **D3** | Input | 列車検知センサー 2 | 外部割り込み（INT1）利用 |
| **D5** | Output | モータードライバ PWM制御 | Timer3 使用 |
| **D22** | Output | 信号機1: 赤色LED | |
| **D23** | Output | 信号機1: 黄色LED | |
| **D24** | Output | 信号機1: 緑色LED | |
